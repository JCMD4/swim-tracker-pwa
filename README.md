# Swim Tracker PWA

Live stroke counting + lap timing at poolside. AI runs on-device via MediaPipe.

**Live app:** https://JCMD4.github.io/swim-tracker-pwa/

Features:
- Real-time stroke counting (free, back, breast, fly)
- Flip-turn detection via pose gap
- Lane tap to track one swimmer in a multi-swimmer pool
- Session history with per-window stroke rate breakdown
- Video recording + download

Source: https://github.com/JCMD4/swim-tracker
